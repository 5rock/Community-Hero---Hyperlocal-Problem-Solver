# Judge Demo Guide

This guide is designed for the presentation. Follow these steps for maximum impact within a 3-minute window.

## 0:00 - 0:30 (The Hook & Landing)
- Open the application. Highlight the modern, premium UI/UX.
- Emphasize the core problem: "Cities are broken, citizens are ignored. We fix that."
- Show the splash screen into the main landing page.

## 0:30 - 1:15 (Citizen Workflow & AI)
- Login as `citizen@example.com`.
- Go to **Report Issue**.
- Type a description in a non-English language (e.g., Hindi) and upload a pothole photo.
- **WOW Moment:** Show how the AI automatically translates it, scores the severity, estimates the cost, and routes it to the "Roads & Transport" department.

## 1:15 - 2:00 (Officer Workflow)
- Logout and Login as `officer@example.com`.
- Notice the distinct, professional dashboard.
- Show the assigned issue from the citizen. Mark it as "Resolved".

## 2:00 - 2:45 (Admin Workflow & Analytics)
- Login as `admin@example.com`.
- Open the Interactive Map to show real-time hot-spots.
- Highlight the Predictive Forecasting AI (e.g., "AI predicts a 35% surge in garbage complaints this weekend").

## 2:45 - 3:00 (Conclusion)
- Show the Leaderboard to emphasize community gamification.
- End on the Profile Page showing impact stats.
